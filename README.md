# CircleChen Plastics Website

Static company website for a US-focused plastic recycling and resin supply business.

## Files

- `index.html` - page content and structure
- `styles.css` - responsive visual design
- `script.js` - sticky header state
- `assets/hero-recycling.png` - generated hero image

## Deployment

This site can be deployed as a static website on Cloudflare Pages, Netlify, Vercel, GitHub Pages, or Porkbun static hosting.
